书单推荐

总结, 最近几年看的一些比较好的书籍, 整理出来. 看书的原则, 基本上国外的书籍, 会优先选择.

编程语言

书名 笔记 备注
Java编程语言
Effective Java 适合有一定Java开发经验的人阅读
深入理解JVM JVM基本最容易理解的阅读的一本书

软件工程

书名 笔记 备注
重构 代码风格的第一本书
卓有成效的程序员 程序员效率提升宝典
程序员修炼之道 从小工到专家之路

操作系统

书名 笔记 备注
鸟哥私房菜 Linux新手入门书籍
Linux内核解析 有一定的难度

个人管理

书名 笔记 备注
优秀到不能被忽略 专家路线 秒杀一切职场书籍
Deep Work 高认知工作, 如何更加深入的思考和工作
慢思考 信息过载时代如何做到深入思考 如何在互联网时代深刻思考和高效工作

关注的作者

有一些作者的思想, 会高度产出,

  • Nassim Taleb

    • 黑天鹅 和 反脆弱 两本书籍的作者, 书中的思想 将哲学, 数学,经济学融会贯通, 看完之后对很多现象将会有更本质的思考.
    • 个人网站: http://www.fooledbyrandomness.com/
    • 最好的两本书:
      • 黑天鹅: https://book.douban.com/subject/2072159/
      • 反脆弱: https://book.douban.com/subject/10545758/

  • Cal Newport

    • Deep Work 和 Be So Good 两本书作者, Newport是一位 MIT 毕业的计算机博士, 善于使用严谨学术的角度来阐述高效学习和职场理论.
    • 个人网站: http://calnewport.com/
    • 最好的两本书:
      • Deep Work: https://www.goodreads.com/book/show/25744928-deep-work?ac=1&from_search=true
      • So Good: https://www.goodreads.com/book/show/13525945-so-good-they-can-t-ignore-you?ac=1&from_search=true

  • Charlie Munger

    • 这位太有名,就不用介绍了, 巴菲特的搭档: 查理芒格, 其一直倡导的 多模型和跨学科的思考方法, 被很多投资和金融领域的人士推崇, 但是这种思想模型不仅仅适用于金融领域, 这更是普世的智慧,关于生活的思考.
    • 个人介绍: https://en.wikipedia.org/wiki/Charlie_Munger
    • 关于Munger几本书:
      • 穷查理宝典: https://book.douban.com/subject/5346110/
